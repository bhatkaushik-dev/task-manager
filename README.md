A simple, clean Task Manager application built with Next.js (App Router) and React.
Users can create, view, update, and delete tasks with local persistence.

Deployed Link - https://task-manager-black-gamma.vercel.app/

Features

1. Create tasks with title & description

2. View tasks grouped by date

3. Mark tasks as completed / pending

4. Delete tasks

5. Persistent storage using localStorage

6. Fully responsive UI

Framework: Next.js 14 (App Router)
Language: TypeScript
UI: React + Tailwind CSS
State Management: React Hooks
Storage: Browser localStorage

Set-up

git clone https://github.com/your-username/task-manager.git
cd task-manager

install dependencies 

npm install
# or
yarn install
# or
pnpm install

Run the Development Server

npm run dev
# or
yarn dev
# or
pnpm dev

visit - http://localhost:3000
